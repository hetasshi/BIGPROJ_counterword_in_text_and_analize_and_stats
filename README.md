  ### 1. –í–≤–æ–¥ —Ç–µ–∫—Å—Ç–∞
  - –ü–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å —Å–º–æ–∂–µ—Ç –≤–≤–µ—Å—Ç–∏ —Ç–µ–∫—Å—Ç –≤—Ä—É—á–Ω—É—é –≤ —Ç–µ–∫—Å—Ç–æ–≤–æ–µ –ø–æ–ª–µ (`TextBox`) –≤ –≥–ª–∞–≤–Ω–æ–º –æ–∫–Ω–µ –ø—Ä–æ–≥—Ä–∞–º–º—ã.  
  - –¢–µ–∫—Å—Ç–æ–≤–æ–µ –ø–æ–ª–µ –±—É–¥–µ—Ç –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞—Ç—å –≤—Å—Ç–∞–≤–∫—É —Ç–µ–∫—Å—Ç–∞ —á–µ—Ä–µ–∑ –∫–æ–ø–∏—Ä–æ–≤–∞–Ω–∏–µ (`Ctrl+V`).  
  - –ú–∞–∫—Å–∏–º–∞–ª—å–Ω—ã–π —Ä–∞–∑–º–µ—Ä —Ç–µ–∫—Å—Ç–∞ –¥–ª—è –∞–Ω–∞–ª–∏–∑–∞ –±—É–¥–µ—Ç —Å–æ—Å—Ç–∞–≤–ª—è—Ç—å –¥–æ **1 –ú–ë** (—á—Ç–æ–±—ã –ø—Ä–æ–≥—Ä–∞–º–º–∞ –Ω–µ —Ç–æ—Ä–º–æ–∑–∏–ª–∞ –ø—Ä–∏ –æ–±—Ä–∞–±–æ—Ç–∫–µ –±–æ–ª—å—à–∏—Ö –æ–±—ä–µ–º–æ–≤).

  ### 2. –ê–Ω–∞–ª–∏–∑ —Ç–µ–∫—Å—Ç–∞
  - –ü—Ä–æ–≥—Ä–∞–º–º–∞ –±—É–¥–µ—Ç –Ω–∞—á–∏–Ω–∞—Ç—å –∞–Ω–∞–ª–∏–∑ —Ç–µ–∫—Å—Ç–∞ –ø–æ—Å–ª–µ –Ω–∞–∂–∞—Ç–∏—è –∫–Ω–æ–ø–∫–∏ **"–ê–Ω–∞–ª–∏–∑–∏—Ä–æ–≤–∞—Ç—å"**.  
  - –ê–Ω–∞–ª–∏–∑ –±—É–¥–µ—Ç –≤–∫–ª—é—á–∞—Ç—å –ø–æ–¥—Å—á–µ—Ç –∏ –≤—ã—á–∏—Å–ª–µ–Ω–∏–µ —Ä–∞–∑–ª–∏—á–Ω—ã—Ö —Å—Ç–∞—Ç–∏—Å—Ç–∏—á–µ—Å–∫–∏—Ö –¥–∞–Ω–Ω—ã—Ö (—Å–º. –ø—É–Ω–∫—Ç 3).

  ### 3. –í—ã–≤–æ–¥ —Å—Ç–∞—Ç–∏—Å—Ç–∏–∫–∏
  –ü–æ—Å–ª–µ –∞–Ω–∞–ª–∏–∑–∞ –ø—Ä–æ–≥—Ä–∞–º–º–∞ –±—É–¥–µ—Ç –æ—Ç–æ–±—Ä–∞–∂–∞—Ç—å —Å–ª–µ–¥—É—é—â–∏–µ –¥–∞–Ω–Ω—ã–µ:  
  - **–û–±—â–µ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Å–ª–æ–≤**: —Å–∫–æ–ª—å–∫–æ —Å–ª–æ–≤ –±—É–¥–µ—Ç –≤ —Ç–µ–∫—Å—Ç–µ. –°–ª–æ–≤–æ–º –±—É–¥–µ—Ç —Å—á–∏—Ç–∞—Ç—å—Å—è –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å —Å–∏–º–≤–æ–ª–æ–≤, –æ—Ç–¥–µ–ª–µ–Ω–Ω–∞—è –ø—Ä–æ–±–µ–ª–∞–º–∏ –∏–ª–∏ –∑–Ω–∞–∫–∞–º–∏ –ø—Ä–µ–ø–∏–Ω–∞–Ω–∏—è (–Ω–∞–ø—Ä–∏–º–µ—Ä, "—Å–ª–æ–≤–æ," –∏ "—Å–ª–æ–≤–æ" –±—É–¥—É—Ç —Å—á–∏—Ç–∞—Ç—å—Å—è –æ–¥–Ω–∏–º —Å–ª–æ–≤–æ–º, –∑–Ω–∞–∫–∏ –ø—Ä–µ–ø–∏–Ω–∞–Ω–∏—è –±—É–¥—É—Ç –∏–≥–Ω–æ—Ä–∏—Ä–æ–≤–∞—Ç—å—Å—è).

  - **–ß–∞—Å—Ç–æ—Ç–∞ —Å–∏–º–≤–æ–ª–æ–≤**: —Å–∫–æ–ª—å–∫–æ —Ä–∞–∑ –±—É–¥–µ—Ç –≤—Å—Ç—Ä–µ—á–∞—Ç—å—Å—è –∫–∞–∂–¥—ã–π —Å–∏–º–≤–æ–ª (–±—É–∫–≤—ã, —Ü–∏—Ñ—Ä—ã, –∑–Ω–∞–∫–∏ –ø—Ä–µ–ø–∏–Ω–∞–Ω–∏—è) ‚Äî –º–æ–∂–Ω–æ –±—É–¥–µ—Ç –æ–≥—Ä–∞–Ω–∏—á–∏—Ç—å—Å—è —Ç–æ–ø-5 –∏–ª–∏ —Ç–æ–ø-10 –¥–ª—è —É–¥–æ–±—Å—Ç–≤–∞.  
  - **–ö–æ–ª–∏—á–µ—Å—Ç–≤–æ –ø—Ä–æ–±–µ–ª–æ–≤**: –æ–±—â–µ–µ —á–∏—Å–ª–æ –ø—Ä–æ–±–µ–ª–æ–≤ –≤ —Ç–µ–∫—Å—Ç–µ –±—É–¥–µ—Ç –ø–æ–¥—Å—á–∏—Ç–∞–Ω–æ.  
  - **–ö–æ–ª–∏—á–µ—Å—Ç–≤–æ –ø—Ä–µ–¥–ª–æ–∂–µ–Ω–∏–π**: –ø—Ä–æ–≥—Ä–∞–º–º–∞ –±—É–¥–µ—Ç —Å—á–∏—Ç–∞—Ç—å –ø—Ä–µ–¥–ª–æ–∂–µ–Ω–∏—è, –æ–ø—Ä–µ–¥–µ–ª—è—è –∏—Ö –ø–æ —Ç–æ—á–∫–∞–º (`.`), –≤–æ—Å–∫–ª–∏—Ü–∞—Ç–µ–ª—å–Ω—ã–º (`!`) –∏ –≤–æ–ø—Ä–æ—Å–∏—Ç–µ–ª—å–Ω—ã–º (`?`) –∑–Ω–∞–∫–∞–º.  
  - **–°—Ä–µ–¥–Ω—è—è –¥–ª–∏–Ω–∞ —Å–ª–æ–≤–∞**: —Å—Ä–µ–¥–Ω–µ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Å–∏–º–≤–æ–ª–æ–≤ –≤ —Å–ª–æ–≤–∞—Ö –±—É–¥–µ—Ç –æ–∫—Ä—É–≥–ª–µ–Ω–æ –¥–æ 1 –∑–Ω–∞–∫–∞ –ø–æ—Å–ª–µ –∑–∞–ø—è—Ç–æ–π.  
  - **–°—Ä–µ–¥–Ω—è—è –¥–ª–∏–Ω–∞ –ø—Ä–µ–¥–ª–æ–∂–µ–Ω–∏—è**: —Å—Ä–µ–¥–Ω–µ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Å–ª–æ–≤ –≤ –ø—Ä–µ–¥–ª–æ–∂–µ–Ω–∏–∏ –±—É–¥–µ—Ç –æ–∫—Ä—É–≥–ª–µ–Ω–æ –¥–æ 1 –∑–Ω–∞–∫–∞ –ø–æ—Å–ª–µ –∑–∞–ø—è—Ç–æ–π.  
  - **–†–∞–∑–º–µ—Ä —Ç–µ–∫—Å—Ç–∞ –≤ –±–∞–π—Ç–∞—Ö**: –æ–±—â–∏–π —Ä–∞–∑–º–µ—Ä –≤–≤–µ–¥–µ–Ω–Ω–æ–≥–æ —Ç–µ–∫—Å—Ç–∞ –≤ –±–∞–π—Ç–∞—Ö –±—É–¥–µ—Ç –≤—ã–≤–µ–¥–µ–Ω (–∏–Ω—Ç–µ—Ä–µ—Å–Ω–∞—è –∏–¥–µ—è!).

  ### 4. –ò–Ω—Ç–µ—Ä—Ñ–µ–π—Å/–≥—É–∏—à–∫–∞

  - –ò–Ω—Ç–µ—Ä—Ñ–µ–π—Å –±—É–¥–µ—Ç –ø—Ä–æ—Å—Ç—ã–º –∏ –∏–Ω—Ç—É–∏—Ç–∏–≤–Ω–æ –ø–æ–Ω—è—Ç–Ω—ã–º, —á—Ç–æ–±—ã –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å —Å—Ä–∞–∑—É –ø–æ–Ω—è–ª, –∫–∞–∫ –ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è –ø—Ä–æ–≥—Ä–∞–º–º–æ–π.

  ### 5. –î–æ–ø —Ñ—É–Ω–∫—Ü–∏–∏, –∫–æ—Ç–æ—Ä—ã–µ —è –±—ã —Ä–µ–∞–ª–∏–∑–æ–≤–∞–ª
